# Bruteforcer

This program is designed to attempt brute force against a simple password.

!(screenshot)[screenshots/screenshot.png]

## Features
- Multithreading
- Hash md5


## Quick Start
```bash
# clone repo
git clone https://github.com/mental0-main/bruteforcer.git

# move to dir
cd bruteforcer

# run
node script.js
```

## Requirements
- git
- node
